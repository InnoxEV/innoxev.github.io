# INNOX Product Spec  
## Ldnio Z4 Universal Plug Adapter

---

## 1. PAGE GOAL

**Primary Goal:**  
Enable BYD V2L users to safely and easily use their car’s power output with multiple plug types anywhere.

**Problem Solved:**  
BYD V2L provides power, but users cannot directly connect many devices due to plug type differences and limited socket availability.

**Why It Matters (Real-World Context):**  
This adapter removes friction when using V2L in:
- travel situations  
- outdoor usage (camping)  
- emergency scenarios  

It transforms the car from a “power source” into a practical, usable energy solution.

---

## 2. TARGET USER

**Primary Users:**
- BYD owners using V2L functionality  

**Secondary Users:**
- Travelers needing multi-standard plug support  
- Outdoor users (camping, road trips)  
- Users needing backup power during outages

### 3.3 How It Works

1. Connect your BYD V2L cable to the car  
2. Plug the Ldnio Z4 adapter into the V2L output  
3. Connect your devices using the universal sockets

**Visual:**  
Simple 3-step diagram or icon-based flow (Car → Adapter → Devices)

**Purpose:**  
Remove confusion and clearly show real usage flow  

---

### 3.4 Compatibility Section

**Explanation:**  
- Works with BYD vehicles that support V2L (Vehicle-to-Load)  
- Converts standard AC output into a universal plug interface  

**Supported Plug Types:**
- UK (3-pin)  
- EU (2-pin)  
- US / AU (2-pin)  

**Important Note:**  
Works with any standard AC output from BYD V2L adapters (region-dependent)

### 3.5 Technical Specifications

| Feature            | Details                     |
|-------------------|----------------------------|
| Model             | Z4                         |
| Rated Voltage     | 110–250V                   |
| Input             | 100–240V, 50–60Hz          |
| Output            | 100–240V, 10A              |
| Rated Current     | 6A                         |
| Material          | Plastic                    |
| Fire Safety       | ABS V0 (Fireproof)         |
| Plug Types        | UK / EU / US / AU          |
| Universal Socket  | Yes (2 sockets)            |
| Size              | 49 × 52 × 58 mm            |

---

### 3.6 Use Cases

- Charging phones and laptops during road trips  
- Powering small devices while camping  
- Using BYD V2L as backup power at home  
- Travel adapter for international plug compatibility  

---

### 3.7 Safety & Notes

- Maximum output: 6A (suitable for small electronics only)  
- Not recommended for high-power appliances:
  - kettles  
  - heaters  
  - large kitchen devices  

- Fire-resistant material (ABS V0) for safer operation  
- Always ensure total load does not exceed rated capacity  
- Suitable for devices up to ~1300W (depending on voltage)
